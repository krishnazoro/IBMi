1. If you need type the RPGLE program in pub400 first of all type the command **wrkmrpd** in the command line.

2. And then select the program there is no space to type the program, so type the **I** on the left side below the FMT the new space will be form.

3. And start the program by typing the `**Free` and and start the second line of the program in last letter of the FREE.

4. If you need to repeat the program line type the **RP** in the numbers like 0006.00.

5. If you need to be copy the program line type the **C** in the number like 0005.00.

6. If you need to copy one line to another line type the **C** in the one program line which you need to copy and type the **A**[After] which line after the line need to be copy.

7.  If you need to copy one line to another line type the **C** in the one program line which you need to copy and type the **B**[before] which line before the line need to be copy.

8. If you need to bulk copy type **CC** on the start line which need to copy and type **CC** on the end line need to be copy and type **A**[After] or **B**[Before] which line need to be copy.

9. If you need to delete the line type the **D** on the number like 0006.00.

10. If you need to bulk delete type the **DD** from the start line to end line on the number like 0007.00.
